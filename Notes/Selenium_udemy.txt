Implicit wait			:	is global wait that applies to all elements and the 
					driver waits for this time 
					and if element is not found still then it throws exception
					and if found earlier then driver without waiting for
					maximum time executes the code

					( advantage	: cleaner, readable code )
Explicit wait			:	used to apply waits to specific elements untill certain
					criteria is not met
